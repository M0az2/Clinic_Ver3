# Clinic Management System

A clinic management desktop application built with C# and ASP.NET as part of a software engineering project.

## Features
- Patient management
- Doctor scheduling
- Appointment booking
- Medical records

## Tech Stack
- **C#** - Programming language
- **ASP.NET** - Framework
- **SQL Server** - Database
- **Windows Forms** - GUI

## How to Run
1. Clone the repo
2. Open `ITIProject.sln` in Visual Studio
3. Update connection string in config
4. Press `F5` to run
